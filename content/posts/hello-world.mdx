欢迎来到我的新博客！

这是我的个人博客的全新版本，使用以下技术栈构建：

- **Next.js 15** - React 框架
- **shadcn/ui** - UI 组件库
- **Tailwind CSS** - 样式框架
- **MDX** - Markdown 扩展

## 特性

这个博客具有以下特性：

1. **响应式设计** - 在各种设备上都能良好显示
2. **日夜主题切换** - 支持明暗模式切换
3. **标签分类** - 按标签浏览文章
4. **全文搜索** - 使用 Fuse.js 进行模糊搜索
